# Sprint Plan – Transfer Credit Match Frontend

## Sprint Pacing

| Sprint   | Focus                                      |
| -------- | ------------------------------------------ |
| Sprint 1 | Project setup + routing only               |
| Sprint 2 | Global layout + navigation                 |
| Sprint 3 | Login & registration UI (no backend)       |
| Sprint 4 | Institution & program selection components |
| Sprint 5 | Course listing with expandable details     |
| Sprint 6 | API integration                            |
| Sprint 7 | Loading indicators & error handling        |

## Important Rule

**Do not implement functionality assigned to future sprints.**

Development must align with sprint reviews and retrospectives. Avoid implementing ahead of the current sprint scope.
